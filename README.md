App to display api response of list of countries.

 - MVVM
 - Kotlin
 - Clean Architecture
 - SOLID principles
 - MAD Developement with UseCases
 - Retrofit networking and cache
 - Single fragment
 - XML Views with constraint layouts for a flat xml hierarchy
 - Manual dependency injection
